# SOC Analyst Tier 1 Portfolio & Practical Labs

##  Project Overview
This repository contains a structured collection of practical incident response, forensic investigation, and threat triage projects completed as part of my training for the SOC Analyst Tier 1 role.

##  Documented Case Studies
Click the technical report links below to view the full investigations:

1. **[Case THM-PHISH-01: Phishing Incident Triage](./phishing-report.md)**
   - **Description:** Simulated Tier 1 Analyst workflow covering email header extraction, validation of SPF/DKIM metrics, and credential harvesting triage using automated open-source threat intelligence (OSINT).

2. **[Case THM-WIN-01: Windows Endpoint Forensics](./windows-endpoint-forensics.md)**
   - **Description:** Digital Forensics and Incident Response (DFIR) investigation tracking threat actor footprints on a compromised Windows host, auditing registry persistence (Run keys, Scheduled Tasks), and analyzing malicious user accounts.

3. **[Case THM-JUICY-01: Juicy Details Log Analysis](./juicy-details-dfir.md)**
   - **Description:** Log analysis and digital forensics investigation of a breached web application (Juicy Shop) to identify attacker methodology, exploited vulnerabilities (Directory Traversal), and exfiltrated data.

##  Combined Skillset Demonstrated
* **Incident Triaging:** Separating True Positives from False Positives based on verifiable indicators (IoCs).
* **Endpoint & Log Forensics:** Auditing Windows Registry, Event Viewer, Apache/Nginx web logs, and tracking threat actor footprints.
* **Cybersecurity Frameworks:** Mapping attack lifecycles using the Cyber Kill Chain and identifying telemetry depth on the Pyramid of Pain.
